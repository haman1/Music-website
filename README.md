## Tech Stack

**Client:** Pug, CSS

**Server:** Node, Express

## Run Locally

Clone the project

```bash
  git clone https://github.com/haman1/Music-website/
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
